# myseckill 秒杀项目
## 代码结构
![image](https://user-images.githubusercontent.com/52461848/167248100-36d89c60-0da9-4ff2-bc5c-40dbde376ec2.png)    
1.  common包：存放一些常用工具类
2.  component包：自己注入的一些Bean    
3.  configuration包：存放一些配置类
4.  controller包：控制类  
5.  service包：业务层 
6.  dao 包: 数据库层  
7.  entity包：实体层
8.  rocket包：RocketMQ的一些代码   
## 运行环境
1.  安装JDK，配置环境变量；    
2.  安装Maven，配置镜像服务器；
3.  配置数据库
